Carambola Tutorial

Welcome to the Carambola Tutorial! This tutorial is designed to give a basic overview of all the features of Carambola.

Please note that this tutorial is not yet complete.

Load and Save

Under the File menu, in the main application, you will find a few options realted to opening, saving and closing a segment. The first is Close, which will close the currently loaded segment, discarding it. The second available option should be Open or Load, which will load a segment into the editor for editing. The last available option that we are concerned with is the option called Save as, which will save a segment to a file that you choose.

See if you can load a segment from Smash Hit, or a segment that someone else has made, even if it's not been made in Carambola.

Objects

Objects can be added and deleted from the Edit menu, using the first five options.

Boxes

Boxes are cubes, with there middle point being at the centre of the cube. You can set the visibility, colour and tile properties to get it to show up when using a tool like MeshBake to bake a mesh file, or turning on the stone hack in the settings.

Obstalces

Obstacles are scripts that build and destroy obstacles; the segment has very little to do with how the obstacle is actually created, besides providing some paramaters.

Prefrences

You can access some prefrences and options to change how carambola works by going into the Edit menu and then clicking Options.
