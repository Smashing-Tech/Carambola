Versions and News

Version 2021-04-10, New Website and Looking for Maintainers

There is a new version of Carambola available, 2021-04-10. This adds Blender-Tools like exporting of hacked stone obstacles.

In addition, I significantly updated the website yesterday and started to write the tutorial for using Carambola.

Lastly, I am looking for people who can develop Carambola more actively than I can to take over the project; I will still try to contribute when I can.

- Knot126

Version 2021-03-27 has been released!

Changes since 2021-03-17:

  • Added basic options
  • Decreased CPU usage by only updating objects when properties are set
  • Fixed crash when saving segment after deleting objects
  • Updated carambola segment extensions
  • Many other smaller fixes and improvements

Version 2021-03-17 has been released!

This is another pre-release of Carambola. This version brings some major changes, none of which should break compatibility:

  • Boxes will now show textures on them
  • Adding the ability to place decals
  • Loading templates to use is now supported, and the default templates are now included
  • Renamed templates menu to tools
  • Updated integer widget to use a spin box
  • Some obstacle parameters now import correctly

Note: In UTC, this was first released on 2021-03-18; however, the release process started before the new day had started in UTC.

Version 2021-03-11 has been released!

This new version brings minor changes, including a rewrite of the UI and property editing system to be more flexible, more traditional names for some menu options and improvements to how adding obstacles work.

While this release should be stable, there are some known issues:

  • It is known that the parameters for obstacles do not load; this feature has been excluded as the system for setting parameters has not been finalised.
  • The android port is not updated with this build as the android build is not any more usable than it was in the last version.

Version 2021-03-06 has been released!

This new version brings minor changes. Changes from 2021-03-05 include:

  • Boxes are coloured like they will be in the game
  • An axis and location changer widget is shown
  • Support for showing XML to a text box
  • Partial fix for android file system issues
