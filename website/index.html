Carambola

Download (EXE)

For Windows(TM), version 2021-07-16

Carambola is a segment editor for Smash Hit, designed to be an evolution from Blender Tools. It is currently in early testing, and it does not fully work at the moment; however, it is capable enough to produce basic Smash Hit segments.

Download

For downloads that can be installed and run without downloading Godot, just click the above button (or the link below it if you're on Android). The editor should not require installation or any special permissions aside from accessing the file system or internal storage. If you need to download an older release, please check the releases on GitHub.

NEW: If you don't want to download any software, there is now an online version of Carambola that can run right in your browser! Make sure to use File -> Get XML to save segments since the normal opening an closing method doesn't work at the moment.

Frequently Asked Questions

Will XYZ ever be supported?/When will XYZ be added?

Maybe. I am just trying to get segment editing fully working, so features are not exactly a priority. As for when the greyed out menus will become available, I am not sure as this project does not follow a fixed release schedule.

Where should I report any bugs?

You can report them on the GitLab project page if you have a gitlab account or send an email to contact-project+smashhit-modding-carambola-bugs@incoming.gitlab.com.

Should I use this tool, or something else for now?

You should use this tool if it will work for you; otherwise, PLEASE use something else for now. This tool is not complete at all and will likely change a lot in the future.

It is open source?

Yes, please see the GitLab project page for the source code. It uses the MIT licence.
