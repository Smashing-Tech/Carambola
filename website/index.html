Version 2021-07-16 for Windows

Important: Carambola is no longer being developed as of 2022-03-16, so there will not be any new updates; however, Carambola will continue to work as it did as of its latest release.

Carambola is a segment editor for Smash Hit, designed to be an evolution from Blender Tools. It is currently in early testing, and it does not fully work at the moment; however, it is capable enough to produce basic Smash Hit segments.

Download information

For downloads that can be installed and run without downloading Godot, just click the above button (or the link below it if you're on Android). The editor should not require installation or any special permissions aside from accessing the file system (or internal storage on Android).

If you need to download an older or Android release, please check the releases on GitHub.

Frequently asked questions

Where should I report any bugs?

You can report them on the GitHub Issues page if you have a GitHub account.

Should I use this tool or something else for now?

You should use this tool if it will work for you; otherwise, please consider using something else.

It is open source?

Up to the release on 2022-07-16, yes, please see the GitHub project page for the source code, not including the Android version 2021-10-17. It uses the MIT licence.
